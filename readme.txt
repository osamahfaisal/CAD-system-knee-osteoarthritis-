Knee Osteoarthritis Diagnostic Application - KOA CAD

Installation Instructions:
1. Run python-3.11.7.exe from the setup folder to install Python 3.11.7. (Ensure to check "Add Python.exe to PATH" during installation.)
2. Double-click the setup.bat file to set up the necessary dependencies. (An internet connection is required.)

Running the Application:
- To run the application, double-click the run.bat file. (No internet connection is required.)

Additional Notes:
- Ensure that Python 3.11.7 is the only version installed on your system before proceeding with the setup.
- Uninstall any other Python versions currently installed on your system.
